# 📊 Server Stats Monitor

Веб-панель для мониторинга состояния сервера в реальном времени. Показывает загрузку CPU, оперативную память, диски, сеть, температуру, позволяет управлять системными службами и просматривать логи.

## 📋 Возможности

- **📈 Дашборд** — CPU (общая загрузка и по ядрам), RAM, диски, сеть, батарея (для ноутбуков)
- **🌡️ Температура CPU** — автоматическое определение датчиков (Intel/AMD)
- **📊 Графики истории** — загрузка CPU и памяти за последние 6 часов (Chart.js)
- **🔄 Управление службами** — запуск, остановка, перезапуск системных служб
- **📋 Логи** — просмотр системных логов и логов конкретных служб
- **🌐 Активные подключения** — список текущих сетевых соединений
- **ℹ️ Системная информация** — хостнейм, ядро, модель CPU, объем RAM, время последней загрузки
- **🔐 Авторизация** — защита паролем, сессии

## 🚀 Быстрый старт

### Требования

- Linux (протестировано на Ubuntu 24.04.4 server)
- Python 3.8+
- pip
- systemd (для управления службами)

### Установка

```bash
# Клонируем репозиторий
git clone https://github.com/ваш-никнейм/server-stats.git
cd server-stats

# Создаем виртуальное окружение
python -m venv venv
source venv/bin/activate

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем
python app.py
